  • Lire 'fandango_scores.csv' dans un dataframe reviews.
  • Slectionner les colonnes suivantes et assigner les au dataframe norm_reviews:
    • 'FILM'
    • 'RT_user_norm'
    • 'Metacritic_user_nom'
    • 'IMDB_norm'
    • 'Fandango_Ratingvalue'
    • 'Fandango_Stars'
  • Afficher les premires lignes de norm_reviews.

import pandas as pd
reviews = pd.read_csv("fandango_scores.csv")
cols = ['FILM', 'RT_user_norm', 'Metacritic_user_nom', 'IMDB_norm', 'Fandango_Ratingvalue', 'Fandango_Stars']
norm_reviews = reviews[cols]
print(norm_reviews[:3])

Diagrammes en barre¶

  • Crer un sous-graphique et assigner l'objet Figure la variable fig et l'objet Axes la variable ax.
  • Gnrer un diagramme en barre avec:
    • left dfinie par bar_positions qui doit prendre les valeurs 1,2,3,4 et 5
    • height dfinie par bar_heights qui doit prendre les 5 colonnes qui nous intressent
    • width fix 0.5
  • Utiliser plt.show() pour afficher le diagramme en barre.
In [8]:
import matplotlib.pyplot as plt
from numpy import arange

num_cols = ['RT_user_norm', 'Metacritic_user_nom', 'IMDB_norm', 'Fandango_Ratingvalue', 'Fandango_Stars']

bar_heights = norm_reviews[num_cols].iloc[0].values
bar_positions = arange(5) + 1

fig, ax = plt.subplots()
ax.bar(bar_positions, bar_heights, 0.5)
plt.show()

num_cols = ['RT_user_norm', 'Metacritic_user_nom', 'IMDB_norm', 'Fandango_Ratingvalue', 'Fandango_Stars']
bar_heights = norm_reviews[num_cols].iloc[0].values
bar_positions = arange(5) + 1
tick_positions = range(1,6)

fig, ax = plt.subplots()
ax.bar(bar_positions, bar_heights, 0.5)
ax.set_xticks(tick_positions)
ax.set_xticklabels(num_cols,rotation=90)

ax.set_xlabel('Sources de notation')
ax.set_ylabel('Note moyenne')
ax.set_title('Moyenne des notes utilisateurs pour le film Avengers: Age of Ultron (2015)')

plt.show()

import matplotlib.pyplot as plt
from numpy import arange
num_cols = ['RT_user_norm', 'Metacritic_user_nom', 'IMDB_norm', 'Fandango_Ratingvalue', 'Fandango_Stars']

bar_widths = norm_reviews[num_cols].iloc[0].values
bar_positions = arange(5) + 1
tick_positions = range(1,6)
fig, ax = plt.subplots()

ax.barh(bar_positions, bar_widths, 0.5)

ax.set_yticks(tick_positions)
ax.set_yticklabels(num_cols)
ax.set_ylabel('Sources de note')
ax.set_xlabel('Note moyenne')
ax.set_title('Note moyenne utilisateurs pour le film Avengers: Age of Ultron (2015)')

plt.show()

import matplotlib.pyplot as plt

fig = plt.figure(figsize=(5,10))

ax1 = fig.add_subplot(3,1,1)
ax2 = fig.add_subplot(3,1,2)
ax3 = fig.add_subplot(3,1,3)

ax1.scatter(norm_reviews['Fandango_Ratingvalue'], norm_reviews['RT_user_norm'])
