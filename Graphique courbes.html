Ajouter une légende¶

  • Modifier le code prcdent pour inclure une lgende prcisant l'anne pour chaque couleur.
    • C'est--dire que la courbe des donnes de 1948 qui utilise la couleur rouge 'red' aura comme intitul dans la lgende '1948'
  • Positionner la lgende dans le coin haut et gauche 'upper left'.
  • Afficher le graphique.
In [13]:
fig = plt.figure(figsize=(10,6))
colors = ['red','blue','green','orange','black']

for i in range(5):
    start_index = i*12
    end_index = (i+1)*12
    subset = unrate[start_index:end_index]
    label = str(1948 + i)
    plt.plot(subset['MONTH'],subset['UNRATE'], c=colors[i], label=label)
plt.legend(loc='upper left')    
plt.show()
